# hotel_manage

## 项目介绍

这是一个使用 vue+vuex+vue-router+axios 写的酒店管理系统 后端是使用 node.js 写的。

## 安装依赖

```
npm install
yarn
```

### 启动本地服务器

```
npm run serve
yarn serve
```

### 打包项目

```
npm run build
yarn build
```

### 检查错误

```
npm run lint
```
